StardustRAG

Streaming · Queue-Backed · Self-Hosted · Docker-Native

1
Pull Stardust RAG
bash
docker pull vickystardust5/stardust:latest
2
Start Dependencies
bash
docker run -d --name rabbitmq -p 5672:5672 rabbitmq:3-management
docker run -d --name qdrant -p 6333:6333 -p 6334:6334 qdrant/qdrant:latest
docker run -d --name ollama -p 11434:11434 ollama/ollama
docker exec -it ollama ollama pull llama3
3
Run Stardust RAG
bash
docker run -it --rm --network="host" --memory="256m" -e AMQP_HOST=127.0.0.1 -e QDRANT_HOST=http://127.0.0.1:6334 -e OLLAMA_HOST=http://127.0.0.1:11434 vickystardust5/stardust:latest
Environment Variables
AMQP_HOST IP / hostname of your RabbitMQ instance
QDRANT_HOST Full URL of your Qdrant server (gRPC port 6334)
OLLAMA_HOST Full URL of your Ollama server (default port 11434)
OLLAMA_MODEL Model name to use for inference (e.g. llama3)
